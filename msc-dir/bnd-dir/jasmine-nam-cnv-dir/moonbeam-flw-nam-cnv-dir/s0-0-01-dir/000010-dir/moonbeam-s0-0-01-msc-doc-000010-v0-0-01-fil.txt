2. Subject Introduction:

	2.1. Flow Naming Convention: 

		2.1.1. The naming convention for flows is based on the type of functionality they provide. For example, the flow named "Sambar User Registration" is specifically designed for handling user registration within the Sambar Web application. Similarly, the flow named "Sambar EKS Cluster" is responsible for creating EKS clusters within the Sambar Web application.
